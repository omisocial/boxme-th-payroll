## Key Architectural Decisions
- **Pages Functions** (not standalone Workers): `webapp/functions/api/[[route]].ts` → same domain, no CORS
- **Cloudflare-only**: D1 (DB) + R2 (files) + KV (sessions) — no Supabase unless D1 >80k writes/day sustained
- **Better-auth on Workers/D1**: cookie-based sessions (HttpOnly, SameSite=Strict)
- **Engine source of truth**: `api/src/engines/compute.ts` — FE engine.ts becomes a stub calling `/api/payroll/compute`
- **Multi-country gate**: `countries.active` bool; inactive countries return 423; default TH
